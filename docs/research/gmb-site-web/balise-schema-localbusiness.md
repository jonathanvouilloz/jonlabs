
Utilisez le balisage de schéma LocalBusiness.

> Ce doit être STRICTEMENT identique à votre fiche GMB.

### **Comment le faire ?**

1. **Choisissez le type précis :** Ne vous contentez pas de "Organization". Utilisez le schéma `LocalBusiness` ou, mieux encore, le sous-type spécifique à votre métier (ex: `Restaurant`, `Dentist`, `Plumber`).
    
2. **Recopiez vos données GMB :** Copiez-collez les informations depuis votre fiche Google Business Profile (nom exact, adresse, numéro de téléphone au même format, horaires). La règle d'or est la **stricte identité**.
    
3. **Intégrez les coordonnées GPS :** Ajoutez les balises `latitude` et `longitude` dans votre code pour que Google puisse faire le lien mathématique parfait avec votre emplacement sur Maps.
    
4. **Testez votre code :** Avant de le publier, passez votre URL dans l'outil de **"Test des résultats enrichis"** de Google pour vérifier qu'il n'y a aucune erreur.
    

### **L'objectif ?**

L'objectif est de passer d'une lecture "visuelle" de votre site par Google à une **lecture "sémantique"**. En utilisant le schéma `LocalBusiness`, vous donnez à Google une fiche d'identité numérique codée qu'il peut traiter instantanément sans risque d'erreur d'interprétation. Cela confirme que l'entité décrite sur votre site est exactement la même que celle qui gère la fiche GMB.

### **De quoi il résulte ? (Le résultat)**

- **Validation de la confiance (Trust) :** Quand Google voit que le code caché de votre site correspond au millimètre près à votre fiche publique, il vous accorde un score de confiance maximal. Il en résulte une stabilité de vos positions, même lors des mises à jour d'algorithme.
    
- **Affichage des "Rich Results" :** Grâce à ce balisage, Google peut afficher des informations comme votre note moyenne ou vos prix directement sous le lien de votre site web dans les résultats classiques. Il en résulte une visibilité accrue qui attire l'œil plus que de simples lignes de texte.
    
- **Maillage Local-Web indissociable :** Le résultat est la création d'un "bloc" d'informations cohérent sur le web. Cela empêche les doublons ou les fiches fantômes de venir parasiter vos résultats, car vous avez clairement défini quelle est la source de vérité pour votre entreprise.
