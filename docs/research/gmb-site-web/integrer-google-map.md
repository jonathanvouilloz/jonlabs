### **Comment le faire ?**

1. **Récupérez le code d'intégration :** Allez sur Google Maps, recherchez votre entreprise, cliquez sur le bouton **"Partager"**, puis choisissez l'onglet **"Intégrer une carte"**.
    
2. **Copiez le code HTML :** Copiez la balise `<iframe...>` générée par Google.
    
3. **Collez-le sur votre page :** Insérez ce code sur votre page "Contact" ou dans le pied de page (footer) de votre site.
    
4. **Assurez-vous qu'il s'agit de VOTRE fiche :** Ne cherchez pas simplement votre adresse, mais bien le **nom de votre établissement**. La carte doit afficher votre épingle avec votre nom, votre note et le nombre d'avis.
    

### **L'objectif ?**

L'objectif est de **prouver à Google la corrélation directe** entre votre site internet et votre fiche physique. En utilisant le code d'intégration officiel, vous créez un lien technique (via l'API Maps) qui confirme que "ce site" appartient bien à "cette fiche". C'est l'un des signaux les plus forts pour valider votre existence réelle et locale.

### **De quoi il résulte ? (Le résultat)**

- **Validation de l'autorité locale :** Vous facilitez le travail de l'algorithme qui n'a plus besoin de "deviner" si vous êtes bien situé là. Il en résulte un renforcement de votre positionnement dans le **Local Pack**.
    
- **Interactivité et Itinéraire :** L'utilisateur peut interagir avec la carte, zoomer, et surtout cliquer sur le bouton "Itinéraire" sans quitter votre site. Il en résulte une transition fluide entre la navigation web et la visite physique dans votre établissement.
    
- **Preuve sociale instantanée :** En intégrant la carte liée à votre fiche, les passants numériques voient immédiatement votre note et vos avis Google. Le résultat est une rassurance immédiate : le client voit que vous existez, que vous êtes situé à proximité et que d'autres clients ont déjà validé vos services.
