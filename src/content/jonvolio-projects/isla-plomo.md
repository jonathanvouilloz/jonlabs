Isla Plomo is a web-based companion app for social deduction board games like The Resistance and Avalon.

## The Problem

Playing Resistance/Avalon requires someone to be the "game master" who assigns roles, which means they can't fully participate. Plus, physical role cards are easy to lose.

## The Solution

A clean, mobile-first web app that:
- Assigns secret roles to all players
- Handles the night phase (eyes closed)
- Tracks mission votes and results
- Works on any device with a browser

## Tech Stack

Built with SvelteKit for the smooth UX, Supabase for real-time sync, and Tailwind for rapid styling.

## Status

Currently live and playable with friends. Planning to add more game modes.
