### H2: Agent métier intégré (multi-outils, données spécifiques) : CHF 6'000 à 15'000
Keywords: tarif agent IA sur mesure, coût automatisation entreprise
Contenu attendu: Détailler ce qui justifie le saut de prix (agent connecté à plusieurs systèmes internes — CRM, ERP, comptabilité —, logique métier plus complexe, mémoire persistante, supervision/contrôles de sécurité). Donner un exemple concret (ex: agent IA pour une régie immobilière qui qualifie les leads ET met à jour le CRM ET planifie les visites — cohérent avec l'article 1). ### H2: Les coûts récurrents que personne ne mentionne (hébergement, tokens, maintenance)
Keywords: coûts cachés automatisation, coût hébergement agent IA

Donner une fourchette mensuelle totale réaliste pour un agent en prod (cohérente avec les chiffres déjà publiés dans hermes-agent-ia-pme : hébergement CHF 5-15/mois, API LLM CHF 10-50/mois — élargir légèrement pour un agent métier plus chargé en usage).

### H2: Forfait fixe ou régie : quel modèle de facturation pour un agent IA
Keywords: forfait vs régie agent IA
Contenu attendu: Expliquer la différence (forfait = prix fermé sur un périmètre défini, plus rassurant pour une PME qui découvre l'IA ; régie = facturation au temps passé, plus adaptée si le périmètre évolue en cours de projet). Donner un avis tranché sur ce qui convient le mieux à une première expérience d'agent IA pour une PME (cohérent avec le ton "opinion forte" de voice.md).

### H2: Pourquoi le prix varie autant d'un projet à l'autre
Keywords: prix agent IA, coût automatisation entreprise
Contenu attendu: Lister les 3-4 facteurs qui font varier le prix dans les fourchettes données (nombre de systèmes à connecter, sensibilité des données, niveau d'autonomie attendu de l'agent, besoin ou non de supervision humaine sur les décisions). Rassurer : ces facteurs sont identifiables dès le premier échange, pas après-coup.

### H2: Comment calculer si l'investissement vaut le coup pour ta PME
