## Note technique (important pour /seo-write)

Cet article DOIT être créé en `.mdx` (pas `.md`) pour permettre l'import du composant calculateur.

```mdx
import CoutDormantCalculateur from '../../components/CoutDormantCalculateur.astro';
```

Le composant `CoutDormantCalculateur.astro` est à créer séparément. Il recevra 3 inputs utilisateur :
- **Panier moyen** (CHF) — valeur d'un client pour le métier
- **Nombre de clients/mois actuellement via le site** (0 si inconnu)
- **Ancienneté du site sans mise à jour** (en mois)

Et affichera :
- Frais fixes annuels estimés (hébergement + domaine = ~150-300 CHF/an)
- Clients manqués/an (basé sur taux de conversion moyen 1.5% × volume estimé)
- Manque à gagner annuel (clients manqués × panier moyen)
- **Total coût de l'inaction sur 1 an / 3 ans**
