## Structure

### Intro (150-200 mots)
Keywords : workflow vs agent IA, automatisation, PME
Contenu : Ouvrir avec une observation terrain directe — ex: "Depuis 6 mois, j'ai au moins une conversation par semaine avec un dirigeant romand qui me pose la même question : 'Est-ce que mon Make, c'est pareil qu'un agent IA ?' La réponse courte : non. La réponse longue, c'est cet article." Poser immédiatement la distinction-clé : un workflow exécute ce que tu lui as dit de faire, un agent décide par lui-même. Annoncer le plan : définitions claires, critères de choix, coûts réels, exemples. Ton direct, pas d'intro en entonnoir générique.

### H2 : Workflows déterministes — ce que font n8n, Make et Zapier
Keywords : automatisation n8n, n8n Make Zapier, workflow déterministe
Contenu : Définition claire : un workflow déterministe suit une logique fixe "SI A ALORS B". Tu définis les règles une fois, le système les applique à chaque fois de la même façon. Exemples concrets pour PME romande : nouvelle facture Stripe → email de confirmation envoyé automatiquement → entrée dans le tableau CRM Google Sheets. Résultats reproductibles à 100%, zéro surprise, zéro décision autonome. Avantages : fiable, auditable, peu coûteux (n8n gratuit self-hosted, Make dès CHF 10/mois, Zapier dès CHF 20/mois), facile à débugger. Limites : ne sait pas s'adapter si le contexte change, bloque si une règle n'est pas prévue. Mentionner que n8n est le quick win à connaître : 210 recherches/mois, LOW competition, open-source.

### H2 : Agents IA autonomes — ce que font Hermes Agent et OpenClaw
Keywords : agent IA autonome, Hermes Agent, OpenClaw, agent IA PME
Contenu : Définition claire : un agent IA autonome reçoit un objectif, pas une règle. Il décide lui-même des étapes pour atteindre l'objectif, peut appeler des outils, mémoriser des informations entre deux sessions, déléguer à des sous-agents. Exemples concrets : "Recherche les 10 derniers devis non-répondus et relance chacun avec un message personnalisé selon son secteur." Un workflow ne peut pas faire ça (il ne personnalise pas en contexte). Présenter les deux agents de référence 2026 : Hermes Agent (Nous Research, $65M levés, 61 000+ GitHub stars, MIT License, février 2026) et OpenClaw (100 000+ GitHub stars, 2M visiteurs au lancement, open source, fondé par Peter Steinberger). Ne pas les vendre — juste expliquer ce qu'ils font depuis l'angle dirigeant. Mentionner aussi Manus et Devin comme représentants de la catégorie (sans entrer dans les détails — les articles S4 et S5 couvrent Hermes et OpenClaw en profondeur). Lien interne vers S4 et S5.

### H2 : La vraie différence en une phrase
Keywords : différence workflow agent IA, workflow automatisation vs agent IA
Contenu : Section courte et percutante (200-250 mots). Synthèse visuelle/conceptuelle de la distinction : workflow = recette de cuisine que tu as écrite toi-même / agent IA = cuisinier qui invente la recette selon ce qu'il y a dans le frigo. Les deux sont utiles, les deux sont complémentaires. La question n'est pas "lequel est meilleur" mais "lequel pour quel usage". Proposer une table de correspondance : WORKFLOW si la tâche est répétitive + toujours identique + bien définie. AGENT IA si la tâche demande du jugement + du contexte + une réponse variable.

### H2 : Quand utiliser un workflow — et quand passer à un agent IA
Keywords : quand utiliser un agent IA, différence workflow agent IA, n8n ou agent IA PME
Contenu : Deux colonnes de cas d'usage, avec exemples concrets pour PME romande. WORKFLOWS (n8n, Make, Zapier) : facturation automatique, rappels clients récurrents, synchronisation CRM, notifications Slack sur nouvel avis Google, génération de rapports hebdo. AGENTS IA (Hermes, OpenClaw) : rédaction de propositions commerciales personnalisées, tri et réponse intelligente aux emails entrants complexes, analyse de données non-structurées (ex: PDF de contrat), veille concurrentielle autonome. Un dirigeant sans dev interne ou prestataire technique → commencer par Make ou Zapier (plus accessible). Un dirigeant avec un dev ou un budget expérimentation → envisager n8n + agent IA pour les tâches à haute valeur ajoutée. Mentionner que les deux catégories sont complémentaires : dans une PME bien structurée, les workflows gèrent la plomberie, les agents gèrent les exceptions.

### H2 : Coûts réels en CHF — ce que tu vas payer
Keywords : agent ia gratuit, coût automatisation PME, tarifs CHF
Contenu : Tableau comparatif clair (pas de jargon). n8n : CHF 0/mois self-hosted, ~CHF 20/mois cloud. Make : CHF 0 free tier (limité), dès CHF 10/mois plan Core. Zapier : CHF 0 free tier, dès CHF 20/mois. Hermes Agent : open source MIT, coût = hébergement (CHF 5-15/mois VPS) + API LLM (CHF 10-50/mois selon usage). OpenClaw : open source, tourne en local sur ta machine — coût essentiellement ton temps de configuration + API LLM si utilisé. Mention importante : les agents IA "gratuits" ont un coût caché en temps de configuration et en tokens LLM. Ce n'est pas zéro — c'est juste différemment facturé. Recommandation honnête : un dirigeant non-technique ne devrait pas configurer un agent IA seul sans accompagnement — le temps perdu coûte plus cher que le prestataire.

### H2 : Par où commencer — décision en 3 questions
Keywords : workflow automatisation vs agent IA, n8n Make Zapier, quand utiliser un agent IA
Contenu : Arbre de décision simple en 3 questions pour le dirigeant : 1) Ta tâche est-elle toujours la même, étape par étape ? → Oui : workflow. Non : potentiellement un agent. 2) Tu as un dev ou un prestataire tech disponible ? → Non : Make ou Zapier. Oui : n8n ou agent IA. 3) La tâche demande-t-elle du jugement ou de la personnalisation contextuelle ? → Oui et tu peux investir le temps de configuration : agent IA. Non ou budget serré : workflow. CTA vers appel découverte : "Tu veux savoir laquelle s'applique à ton cas ? Réserve un appel gratuit." Lien vers le pillar (guide complet sur l'automatisation PME suisse).

### FAQ (3-4 questions issues des PAA)
Keywords : long-tail FAQ, agent ia c'est quoi
Questions à couvrir :
- "Quelle différence entre n8n et Make ?" — réponse courte : même catégorie (workflow déterministe), différences sur l'hébergement et la complexité technique
- "Quand utiliser un agent IA plutôt qu'un workflow ?" — réponse directe avec les 3 critères
- "Les agents IA peuvent-ils remplacer les workflows ?" — non, complémentaires — explication courte
- "Hermes Agent ou OpenClaw, lequel choisir pour une PME ?" — pointer vers les articles S4 et S5

### Conclusion + CTA (150 mots)
Keywords : workflow automatisation, agent IA PME, appel découverte
Contenu : Synthèse en 3 lignes : workflows pour la plomberie répétitive, agents IA pour les tâches qui demandent du jugement — les deux ont leur place dans une PME bien structurée. Pas besoin de tout changer du jour au lendemain. CTA principal : "Réserver un appel découverte" (Calendly). CTA secondaire : "Discuter de votre projet" (/contact). Mention de la suite naturelle : le guide complet automatisation PME suisse (pillar) si le lecteur veut aller plus loin.
