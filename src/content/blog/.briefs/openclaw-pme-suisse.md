## Structure

### Intro (180-220 mots)
Keywords : OpenClaw PME Suisse, agent IA autonome, agent ia gratuit
Contenu : Ouvrir avec les chiffres qui font leur effet sans hype — ex: "OpenClaw a atteint 100 000 étoiles sur GitHub et 2 millions de visiteurs en une semaine lors de son lancement. Pour un outil open source inconnu six mois avant, c'est un signal difficile à ignorer." Poser l'enjeu de l'article : la plupart des contenus sur OpenClaw sont soit en anglais, soit écrits par des développeurs pour des développeurs. Ce guide est pour un dirigeant de PME qui veut une réponse directe : est-ce que c'est quelque chose que je peux utiliser dans mon entreprise, combien ça me coûte, et est-ce que c'est mieux ou différent de Hermes Agent dont on parle aussi beaucoup ? Annoncer le plan : ce qu'OpenClaw fait, qui l'a créé, les cas d'usage PME, les coûts en CHF, et la comparaison frontale avec Hermes Agent et le reste du marché.

### H2 : OpenClaw, c'est quoi — en langage de dirigeant
Keywords : OpenClaw c'est quoi, agent IA autonome, OpenClaw WhatsApp Telegram
Contenu : Explication sans jargon technique. OpenClaw est un assistant IA persistant qui vit sur ta machine (Mac, PC ou serveur) et qui s'intègre aux canaux de communication que tu utilises déjà — WhatsApp, Telegram, SMS, email — pour être accessible partout, comme un vrai collaborateur. La différence fondamentale avec ChatGPT ou Claude : OpenClaw mémorise tout. Chaque conversation, chaque tâche, chaque contexte projet. Quand tu reviens 3 semaines plus tard, il sait où tu en étais. Il peut aussi prendre des actions autonomes : rechercher des informations, rédiger, envoyer des messages, gérer des tâches — pas seulement répondre à des questions. La métaphore utile : imagine un assistant de direction qui t'accompagne sur tous tes appareils, se souvient de tout, et peut agir à ta place sur des tâches bien définies. Ce n'est pas un chatbot. C'est un agent. Lien interne vers S1 pour la distinction workflow vs agent IA.

### H2 : Qui est derrière — Peter Steinberger et la genèse d'OpenClaw
Keywords : Peter Steinberger OpenClaw, agent IA autonome, OpenClaw open source
Contenu : Présentation de Peter Steinberger (@steipete) — fondateur de PSPDFKit (renommé Nutrient), une entreprise de composants PDF que des milliers de développeurs utilisent dans le monde entier. Quelqu'un avec un historique sérieux dans l'open source et le développement d'outils professionnels. OpenClaw n'est pas un side project de hobbyiste — c'est un projet porté par quelqu'un qui a construit et vendu des outils adoptés à grande échelle. Ce contexte compte : une équipe avec une culture de produit sérieuse (documentation, stabilité, expérience utilisateur) derrière un outil open source. 100 000+ GitHub stars. 2 millions de visiteurs en une semaine au lancement. Pas un bruit de fond dans la communauté — un signal clair. Préciser cependant : open source ne signifie pas "entreprise avec financement institutionnel" comme Nous Research. Le projet vit par sa communauté et la vision de son fondateur — ce qui a des implications sur la durabilité à long terme à surveiller.

### H2 : Ce que ça fait concrètement pour une PME — 5 cas d'usage
Keywords : OpenClaw cas usage PME, agent IA entreprise, OpenClaw WhatsApp
Contenu : 5 cas d'usage avec résultats attendus honnêtes et limites pour chaque :
1. **Assistant de direction toujours disponible** : tu lui envoies un message WhatsApp depuis ton téléphone en déplacement — il cherche, rédige, rappelle des infos sur tes projets en cours. Résultat : accès à ton assistant sans ouvrir un laptop. Limite : nécessite une connexion et que l'instance tourne en permanence (machine ou serveur).
2. **Rédaction contextuelle de propositions** : tu lui donnes le brief client par message vocal ou texte — il produit le premier draft de ta proposition en 5 min, avec ton contexte mémorisé depuis les semaines précédentes. Résultat : 70-80% du temps de rédaction récupéré sur le premier draft. Limite : relecture humaine toujours nécessaire sur les éléments de prix et les engagements contractuels.
3. **Tri et priorisation des emails entrants** : il lit ta boîte, identifie les urgences, classe et résume les threads complexes. Résultat : 1-2h/jour récupérées sur le tri. Limite : les emails sensibles (contrats, conflits, RH) nécessitent toujours validation humaine.
4. **Suivi de projets en parallèle** : il garde le contexte de chacun de tes projets actifs et peut te donner un état des lieux à tout moment. Résultat : fini les "où j'en étais ?" le lundi matin. Limite : nécessite une discipline d'alimentation en informations (dire à l'agent ce qui change).
5. **Veille et synthèse hebdomadaire** : il surveille les sources que tu lui confies, filtre et résume une fois par semaine. Résultat : 2-3h/semaine économisées sur la veille manuelle. Limite : la qualité de la synthèse dépend de la qualité des sources configurées.

### H2 : Ce que ça coûte vraiment — décomposition CHF
Keywords : OpenClaw coût installation, agent ia gratuit, coût agent IA PME
Contenu : Décomposition honnête des coûts réels :
1. **OpenClaw lui-même** : open source, gratuit à télécharger et utiliser.
2. **Hébergement** : tourne en local sur ta machine (MacBook, PC) — coût = 0 si tu es prêt à laisser ta machine allumée. Ou VPS cloud CHF 5-15/mois pour une instance dédiée accessible 24h/24.
3. **API LLM** : OpenClaw appelle un modèle de langage pour raisonner. Options : Claude (Anthropic) CHF 10-60/mois selon usage, OpenAI GPT CHF 10-50/mois, ou modèle local via Ollama (coût = GPU/machine). L'option Claude est recommandée pour la qualité sur les tâches de rédaction et d'analyse.
4. **Configuration initiale** : l'interface est plus accessible que Hermes Agent, mais une configuration sérieuse (intégrations WhatsApp, paramétrage des outils) prend 2-8h. Avec prestataire : CHF 200-800.
5. **Maintenance** : outil actif en développement = mises à jour fréquentes. Prévoir 30-60 min/mois de suivi.
Budget réaliste PME avec prestataire : CHF 500-1'200 première année (setup + API + hébergement optionnel). Moins lourd que Hermes Agent sur la configuration initiale pour une utilisation individuelle. Plus lourd si on veut un déploiement multi-utilisateurs.

### H2 : OpenClaw vs Hermes Agent — comparaison directe et verdict
Keywords : OpenClaw vs Hermes Agent, comparaison agents IA, lequel choisir
Contenu : Tableau comparatif sur 7 critères + recommandation par profil :

| Critère | OpenClaw | Hermes Agent |
|---|---|---|
| Fondateur | Peter Steinberger (PSPDFKit) | Nous Research ($65M levés) |
| GitHub stars | 100 000+ | 61 000+ |
| Hébergement | Local (Mac/PC) ou VPS | VPS requis |
| Accès mobile | WhatsApp, Telegram, SMS natif | Interface web ou API |
| Point fort | Expérience utilisateur, accès multichannel | Architecture multi-agents, tâches complexes d'entreprise |
| Idéal pour | Dirigeant solo, assistant personnel omnicanal | PME avec workflow multi-agents, besoin de mémoire structurée |
| Coût setup | CHF 200-800 | CHF 400-1'200 |

Verdict par profil :
- **Dirigeant solo ou TPE, besoin d'un assistant personnel** → OpenClaw : accès mobile direct, setup plus léger, expérience utilisateur supérieure.
- **PME 5-15 personnes, workflows d'équipe, tâches multi-agents complexes** → Hermes Agent : architecture plus adaptée aux déploiements d'entreprise.
- **Tu veux les deux** → possible et complémentaire : OpenClaw pour l'usage quotidien personnel, Hermes pour les workflows opérationnels.
Lien interne vers S4 (guide complet Hermes Agent) pour ceux qui veulent approfondir.

### H2 : OpenClaw vs le reste du marché — Manus, Devin, Claude Code
Keywords : Manus agent IA, comparaison agents IA 2026, OpenClaw marché
Contenu : Vue d'ensemble du marché des agents IA autonomes 2026 — OpenClaw n'est pas seul :
- **Manus** : agent multi-modal de ByteDance (liste d'attente ouverte), fort sur les tâches web complexes (recherche, rédaction, analyse), interface web cloud — pas open source. Idéal pour les tâches de recherche intensive. Limite : dépendance cloud, pas de mémoire persistante de même niveau.
- **Devin** (Cognition Labs) : agent spécialisé pour le code. Prix : $500/mois. Clairement orienté développeurs — hors scope PME non-technique.
- **Claude Code / ChatGPT Projects** : des assistants avec mémoire contextuelle mais pas des agents autonomes — ils ne prennent pas d'actions proactives, ils attendent tes questions.
Positionnement d'OpenClaw dans ce marché : le seul agent open source avec accès multichannel natif (WhatsApp, Telegram) qui tourne en local — différenciateur fort pour les PME qui veulent garder leurs données en local et leur assistant accessible depuis n'importe quel appareil.

### FAQ (4 questions issues des PAA)
Keywords : long-tail FAQ, OpenClaw c'est quoi
Questions à couvrir :
- "OpenClaw est-il vraiment gratuit ?" — réponse directe : open source = gratuit, mais API LLM + config = coûts réels
- "Comment installer OpenClaw sur Mac ?" — réponse courte : téléchargement GitHub, configuration LLM (Claude ou OpenAI recommandé), intégration WhatsApp optionnelle. Recommander prestataire pour PME non-technique.
- "OpenClaw vs Hermes Agent : lequel choisir ?" — synthèse du tableau comparatif selon profil
- "Peut-on utiliser OpenClaw pour une entreprise de 5-10 personnes ?" — réponse honnête : oui pour les usages individuels du dirigeant, avec des limites pour le déploiement multi-utilisateurs (Hermes Agent plus adapté dans ce cas)

Il est différent de Hermes Agent (moins complexe côté entreprise, plus accessible côté usage personnel), différent de Manus (local vs cloud), différent de ChatGPT (action vs conversation). Ce n'est pas encore un outil grand public — c'est un outil pour les dirigeants qui veulent prendre de l'avance.
