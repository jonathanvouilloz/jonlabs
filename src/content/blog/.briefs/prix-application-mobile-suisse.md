### H2 : Ce qui fait vraiment varier le prix d'une app
- Complexité fonctionnelle (CRUD simple vs logique métier complexe)
- Nombre de plateformes (iOS only, Android only, cross-platform)
- Design sur mesure vs templates
- Backend et API (app standalone vs connectée)
- Intégrations tierces (paiement, notifications push, géolocalisation)

### H2 : Fourchettes de prix par type d'application (en CHF)

**Tableau comparatif :**
| Type d'app | Description | Fourchette CHF |
|---|---|---|
| App vitrine / catalogue | Présente des produits/services, pas de logique métier | 5 000 – 12 000 CHF |
| App métier simple | Formulaires, suivi, gestion interne basique | 12 000 – 30 000 CHF |
| App marketplace / SaaS | Comptes utilisateurs, paiements, back-office | 35 000 – 100 000 CHF+ |
| MVP (version minimale) | Périmètre réduit pour valider un concept | 8 000 – 25 000 CHF |

### H2 : Freelance suisse vs agence suisse vs offshore — comparatif prix
- Freelance CH : 80–150 CHF/h — avantages (flexibilité, contact direct, spécialisation) et limites
- Agence CH : 150–250 CHF/h — avantages (structure, garanties, équipe) et limites (coût overhead)
- Offshore (Inde, Ukraine, Maroc) : 15–40 EUR/h — risques réels à mentionner (communication, qualité, fuseaux, IP)
- Tableau de comparaison

### H2 : Les coûts cachés à anticiper
- Frais App Store (99 USD/an Apple, 25 USD une fois Google)
- Maintenance annuelle (bug fixes, mises à jour OS) : 10–20% du coût initial/an
- Backend / serveur (hébergement API, base de données) : 50–300 CHF/mois
- Mise à jour iOS/Android chaque année (nouvelles API requises)
- Traduction et localisation si marché multilingue

### H2 : Ce qu'on peut faire avec 10k, 30k, 80k CHF
- **10 000 CHF** : MVP cross-platform basique (Flutter/RN), 3-4 écrans, sans backend complexe
- **30 000 CHF** : App métier complète, back-office, authentification, 2 plateformes
- **80 000 CHF** : App marketplace, paiements intégrés, back-office admin, API propre

### H2 : Comment obtenir un devis fiable
- Rédiger un cahier des charges avant de contacter un dev
- Demander un découpage par fonctionnalités, pas un prix global
- Exiger un devis fixe ou un budget plafond avec jalons
