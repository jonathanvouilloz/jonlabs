## Structure

### Intro (~120 mots, sans H2)
Keywords : mon site web ne fonctionne pas
Contenu : hook direct franco-suisse ("Si tu sens que ton site ne sert à rien, t'as probablement raison"). Puis promesse concrète : 15 questions fermées, 4 minutes, score sur 15, diagnostic par palier, plan lundi matin. Planter le fait que "mon site web ne fonctionne pas" est la phrase exacte que les clients Jon Labs utilisent. Pas de CTA, pas de TL;DR chiffré "à l'ère du numérique".

### TL;DR (bloc après intro)
Keywords : site web qui ne convertit pas, diagnostic site internet
Contenu : 3 puces courtes.
- V1 chiffre : "7 indépendants romands sur 10 ont un site qui a arrêté de travailler. Voici comment savoir si c'est ton cas en 4 minutes."
- V2 question / méthode : "15 questions oui/non → score sur 15 → 3 paliers de diagnostic → plan d'action par palier."
- V3 take-away : "Version PDF dispo en bas de page pour remplir au stylo ou partager."

### H2 : Pourquoi cette checklist (et à qui elle s'adresse)
Keywords : mon site web ne fonctionne pas, signes site web inactif
Contenu : définir "site qui dort" = en ligne mais muet (personne ne le trouve, personne n'y reste, personne ne t'appelle). Profil visé = indépendant ou PME romande, site > 12 mois, impression vague que "ça ne ramène rien". Préciser que la checklist ne remplace pas un audit pro mais évite d'en payer un tant que le minimum n'est pas fait. **Placer ici le premier lien interne** vers le pilier : `/blog/visibilite-site-internet-2026` — ancre : "le guide complet sur les sites qui travaillent vraiment".

### H2 : Les 15 signes (mode d'emploi)
Keywords : signes site web inactif, checklist site web professionnel
Contenu : 2 phrases de mode d'emploi (1 point par "oui", sois honnête, total sur 15), puis les 5 H3 thématiques qui regroupent les 15 signes exacts. Chaque signe = titre en **gras** + 1-2 phrases d'explication concrète (pas de théorie).

#### H3 : Visibilité Google (signes 1-3)
Keywords : comment savoir si Google ignore mon site
- Signe 1 — Tu n'as jamais ouvert Google Search Console (ou tu ne sais pas ce que c'est). Explication : c'est le tableau de bord gratuit qui dit qui te trouve et sur quoi.
- Signe 2 — Tu tapes le nom exact de ton activité + ta ville sur Google et tu n'apparais pas en page 1. Exemples : "graphiste Lausanne", "ostéopathe Vevey", "avocat Genève divorce".
- Signe 3 — Ta fiche Google Business n'a pas bougé depuis > 6 mois (ou tu n'en as pas). Pas de nouvelles photos, pas de post, pas de réponse aux avis.

#### H3 : Contenu (signes 4-6)
Keywords : signes site web inactif
- Signe 4 — Dernier article/actualité > 6 mois (ou jamais rien publié).
- Signe 5 — Pages "services" < 300 mots et toutes identiques.
- Signe 6 — Aucun blog, aucune ressource, aucune FAQ publique.

#### H3 : Technique (signes 7-9)
Keywords : site web qui ne convertit pas
- Signe 7 — Pas de HTTPS (pas de cadenas dans la barre d'adresse).
- Signe 8 — Illisible sur téléphone (texte minuscule, boutons trop petits, images qui débordent). Rappel : > 60 % du trafic romand est mobile.
- Signe 9 — Plus de 3 secondes d'affichage sur 4G moyenne. Renvoi vers PageSpeed Insights.

#### H3 : Conversion (signes 10-12)
Keywords : site web qui ne convertit pas, site web qui ne rapporte rien
- Signe 10 — Aucun CTA clair au-dessus de la ligne de flottaison (pas de bouton "prendre contact" / "demander un devis" visible sans scroller).
- Signe 11 — Formulaire de contact jamais testé depuis > 6 mois.
- Signe 12 — Aucun outil de suivi (ni GA4, ni Plausible, ni Fathom).

#### H3 : Autorité et E-E-A-T (signes 13-15)
Keywords : audit site web indépendant
- Signe 13 — Aucune page "À propos" réelle (ou générique, sans photo, sans histoire, sans preuve).
- Signe 14 — Aucun avis client visible sur le site (ni témoignage, ni capture Google, ni logo client).
- Signe 15 — Aucun auteur identifié (pas de nom, pas de photo, pas de lien LinkedIn).

### H2 : Interprétation — ton site dort-il vraiment ?
Keywords : diagnostic site internet, site web qui ne rapporte rien
Contenu : tableau récap des 3 paliers + 1 paragraphe diagnostic par palier.

| Score | Diagnostic | État |
|---|---|---|
| 0-5 | Site réveillé | Optimisations à la marge |
| 6-10 | Site en sommeil | Potentiel gros gaspillé |
| 11-15 | Site mort clinique | Tu perds des clients chaque semaine |

- **0-5** : site réveillé → recommandation : renforcer ce qui existe, relire la checklist tous les 6 mois.
- **6-10** : site en sommeil → zone la plus fréquente, potentiel de 50-70% gaspillé, **placer ici le 2ᵉ lien interne** vers `/blog/0a30joursgooglevisible` — ancre : "le plan 30 jours pour redevenir visible".
- **11-15** : site mort clinique → ton direct, tu paies l'hébergement d'un site qui travaille contre toi, 2 options (temps ou délégation).

### H2 : Plan d'action selon ton score
Keywords : audit site web indépendant, site web qui ne convertit pas
Contenu : 1 plan par palier.
- **Score 0-5** : 1 action par trimestre (1 article, refresh À propos, 3 avis Google).
- **Score 6-10** : plan 30 jours en 4 semaines (technique → GSC+GMB → article pilier → CTA/formulaire). Re-lien vers `/blog/0a30joursgooglevisible` (ancre différente, ex : "le plan 30 jours étape par étape").
- **Score 11-15** : 3 cas de figure.
  - Cas 1 : site > 4 ans → refonte probablement plus rapide que rustine. **Placer ici le 4ᵉ lien** vers `/services/refonte-site-web` — ancre : "demander un audit personnalisé".
  - Cas 2 : site récent monté sur Lovable / Bolt / Claude Code → beau et invisible. **Placer ici le 3ᵉ lien** vers `/blog/creer-site-vitrine-ia-visibilite-google` — ancre : "les sites montés sur Lovable, Bolt ou Claude Code stagnent pour la même raison".
  - Cas 3 : site < 2 ans classique → sauvable avec accompagnement méthodique.

Conclusion du H2 : ramener au pilier (lien en deuxième occurrence, ancre différente, ex : "comment tenir un site dans la durée").

### H2 : FAQ
Keywords : pourquoi mon site n'amène aucun client, comment savoir si Google ignore mon site

4. Est-ce qu'un site qui dort peut se réveiller sans tout refaire ?
5. Combien de temps avant de voir un résultat après avoir corrigé les signes ?
