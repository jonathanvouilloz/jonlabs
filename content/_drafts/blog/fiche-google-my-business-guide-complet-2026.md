## Structure H2/H3

### H2 : Qu'est-ce qu'une fiche Google My Business en 2026 (et pourquoi "GMB" est un nom qui n'existe plus)
Keywords : fiche google my business, fiche d'établissement google, google business profile francais, mon entreprise plus
Contenu : Clarifier le rebrand Google My Business → Google Business Profile (GBP) depuis 2021, et pourquoi tout le monde dit encore "GMB". Expliquer ce qu'est concrètement la fiche : panneau d'affichage dans le Knowledge Graph de Google, visible sur Search + Maps. Définir les 3 types (commerce physique / zone de service / SaaS/en ligne). Mentionner que c'est gratuit. Poser la distinction critique : fiche GMB ≠ site web, les deux se renforcent mais ne se remplacent pas.

### H2 : Pourquoi c'est l'actif local le plus sous-estimé d'une PME en 2026
Keywords : fiche google business, signaux de ranking local, local pack
Contenu : Partir du chiffre Whitespark LSRF 2026 — GBP signals = 32% du Local Pack. Expliquer le Local Pack (les 3 résultats Maps en haut de page) et pourquoi il écrase le CTR sur mobile. Introduire le dual-algorithme : Local Pack (GBP domine à 32%) vs AI Search Visibility (le site web est plus important à 24% On-Page). Conclusion actionnable : négliger la fiche = rater le Local Pack ; négliger le site = rater l'AI Search. Les deux sont nécessaires.

### H2 : Le cycle de vie d'une fiche Google : créer → optimiser → animer → mesurer
Keywords : créer fiche google my business, optimiser fiche google my business, gérer mon entreprise
Contenu : Présenter le framework en 4 étapes comme une structure pillar qui lie vers les 7 satellites du cluster. **Créer** (étapes setup, vérification — lien vers M1.03 creer-fiche-google-my-business-etape-par-etape). **Optimiser** (catégorie, description, photos, attributs — lien vers M1.04 optimiser-fiche-google-my-business-checklist-2026). **Animer** (posts, avis, réponses — liens vers M1.05, M1.06, M1.07). **Mesurer** (GBP Insights, appels, itinéraires). Chaque mini-section = 2-3 phrases + lien satellite.

### H2 : Le mythe des backlinks vers la fiche GBP (démystifié avec les données 2026)
Keywords : backlinks fiche google, link signals local pack
Contenu : Reprendre l'analyse `analyse-backlinks-gbp-vs-site.md`. Les backlinks vers GBP ne transmettent pas de PageRank — GBP n'est pas une page web indexable, c'est une entité dans le Knowledge Graph. Ce qui marche : backlinks locaux vers le site (8% des signaux LSRF 2026), citations NAP cohérentes (6-8%), NOT des liens pointant vers la fiche elle-même. Démonter le mythe "Google Stacks". Section courte (~250 mots), lien vers M3.19 pour la version complète.

### H2 : Les vrais signaux de ranking en 2026 (et leur poids réel)
Keywords : signaux de ranking google my business, ranking local 2026, whitespark
Contenu : Tableau complet des signaux Whitespark LSRF 2026 : GBP Signals 32%, Review Signals 20%, On-Page 15%, Behavioral 9%, Link Signals 8%, Citations 6-8%, Personalization 5-6%, Social 3-4%. Expliquer brièvement chaque groupe. Insister sur ce que ça change concrètement : les avis = 20%, donc répondre à 100% des avis est un levier direct de ranking. Les photos, les posts, les attributs = GBP Signals (32%) — tout ça compte. Renvoi vers M1.04 pour la checklist actionnable.

### H2 : Les erreurs qui plombent une fiche (et comment les diagnostiquer)
Keywords : fiche google my business suspendue, erreurs google my business, fiche invisible
Contenu : Lister 5-6 erreurs typiques avec diagnostic rapide : (1) NAP incohérent entre GBP et site web, (2) catégorie principale mal choisie, (3) fiche non vérifiée ou suspicion de doublon, (4) fiche suspendue sans le savoir — lien vers M1.08 pour le recours complet, (5) aucun post depuis >90 jours (signal de fraîcheur mort), (6) 0 réponse aux avis négatifs. Format diagnostic : "Symptôme → Cause probable → Fix rapide".

### H2 : Outils pour suivre la performance de ta fiche
Keywords : mesurer performance google my business, GBP insights, outils suivi local
Contenu : Outils gratuits : GBP Insights natif (impressions, clics, appels, itinéraires, photos vues). UTM sur les liens de posts pour tracker dans Google Analytics. Outils payants mentionnés sans promotions : LocalRanker (grid tracking), Whitespark Local Citation Finder, BrightLocal. Donner 3 métriques à surveiller chaque mois : clics vers site, demandes d'itinéraire, appels directs depuis GBP. Conclure sur la cadence de suivi mensuelle recommandée.
