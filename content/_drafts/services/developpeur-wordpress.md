## Structure (7 sections page service — 3 H2 = 3 intents)

### Hero
Keywords: freelance wordpress geneve
Contenu: H1 + sous-titre qui parle aux DEUX profils (je veux un site WP / mon site WP me pose problème) + CTA. **Quick win GSC : Google montre déjà la homepage en page 4 sur cette query — cette page doit la remplacer.**

### H2: Création de site WordPress
Keywords: création site wordpress, développeur wordpress
Contenu: quand WordPress est le bon choix (blog actif, édition fréquente par le client, budget contenu, plugins métier). Thème sur-mesure léger, pas de page builder usine à gaz. Performance et SEO natifs dès le départ.

### H2: Refonte de site WordPress lent ou vieilli
Keywords: refonte site wordpress
Contenu: le pain point n°1 du persona (site qui rame, design 2015, plugins entassés, prestataire injoignable). Diagnostic → refonte dans WordPress (nettoyage, thème refait, perf) quand la base est saine. Mentionner l'audit gratuit (CTA chaud).

### H2: Migration WordPress vers une stack moderne
Keywords: migration wordpress (50/mois CH — top volume du cluster)
Contenu: quand WordPress n'est plus le bon outil (site vitrine sans blog actif, coûts de maintenance/hébergement/plugins récurrents, failles sécu). Migration vers Astro & co : contenu préservé, redirections 301, SEO conservé, maintenance quasi nulle. Pont assumé vers le sur-mesure — l'angle que PERSONNE n'occupe sur la SERP.

### H2: Méthode & maintenance
Keywords: maintenance wordpress, expert wordpress
Contenu: process en 4 étapes + ce qui se passe après livraison (formation à l'édition, contrat de maintenance optionnel SANS dépendance — contraste avec le pain point "prestataire qui ne répond plus").

### H2: Tarifs

Contenu: fourchettes CHF transparentes (gap : aucun concurrent CH n'affiche de prix — ni Mäder ni Pixana). Création / refonte / migration. Cohérent avec /tarifs.
